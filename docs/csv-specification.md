# CSV仕様書: 業務管理システム (work-system-project)

本システム群でモジュール間連携に使用するCSVファイルの仕様および共通コード体系を定義します。

## 共通コード体系

各モジュールで共通して使用するコードの定義です。

| コード名 | 概要 | 形式 | 例 |
| :--- | :--- | :--- | :--- |
| **employee_code** | 社員を一意に識別するコード | 英数字 10桁以内 | `E00123` |
| **department_code** | 部署を一意に識別するコード | 英数字 4桁 (D+3桁) | `D001` |
| **product_code** | 製品を一意に識別するコード | 英数字 20桁以内 | `PROD-A-001` |
| **work_code** | 作業種別を一意に識別するコード | 英数字 10桁以内 | `W-ASSEMBLY` |

## CSVファイル仕様

### 1. 社員管理CSV (Employee Master)
社員管理システムから出力され、原価計算システム等で利用されます。

| 項目名 | 必須 | 形式 | 備考 |
| :--- | :--- | :--- | :--- |
| employee_code | ○ | 文字列 | 共通コード参照 |
| name | ○ | 文字列 | 社員名 |
| furigana | ○ | 文字列 | 社員名（フリガナ） |
| department_code | ○ | 文字列 | 共通コード参照 |
| position | - | 文字列 | 役職 |
| hourly_wage | ○ | 数値 | 社員ごとの時給 (円) |

### 2. 製品管理CSV (Product Master)
製品管理システムから出力され、原価計算システム等で利用されます。

| 項目名 | 必須 | 形式 | 備考 |
| :--- | :--- | :--- | :--- |
| product_code | ○ | 文字列 | 共通コード参照 |
| name | ○ | 文字列 | 製品名 |
| unit_price | ○ | 数値 | 製品単価 (円) |
| category | - | 文字列 | 製品カテゴリ |

### 3. 作業実績CSV (Performance Record)
作業実績管理システムから出力され、原価計算システム等で利用されます。

| 項目名 | 必須 | 形式 | 備考 |
| :--- | :--- | :--- | :--- |
| work_date | ○ | 日付 (YYYY-MM-DD) | 作業実施日 |
| employee_code | ○ | 文字列 | 共通コード参照 |
| department_code | ○ | 文字列 | 共通コード参照 |
| work_code | ○ | 文字列 | 共通コード参照 |
| product_code | ○ | 文字列 | 共通コード参照 |
| work_hours | ○ | 数値 (小数点可) | 作業時間 (時間) |
| quantity | ○ | 整数 | 製造/処理数量 |

## 共通フォーマット規定
- **文字エンコーディング:** UTF-8 (BOMあり推奨)
- **区切り文字:** カンマ (,)
- **改行コード:** CRLF
- **ヘッダー行:** 1行目はヘッダー行（項目名）とする
