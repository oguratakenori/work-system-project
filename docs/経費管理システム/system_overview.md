# 経費管理システム システム概要

## システム目的
社内の経費実績（月次および日次）を効率的に登録・管理し、分析や報告のためのデータ出力（CSV）を可能にすること。

## システム構成
- **ポート番号**: 5004
- **データベース**: `expense_management.db` (SQLite)
- **識別色**: パープル (#8e44ad)

## テーブル設計

### accounts (勘定科目)
| カラム名 | 型 | 説明 |
| :--- | :--- | :--- |
| id | Integer | プライマリキー |
| code | String | 科目コード |
| name | String | 科目名 |
| is_visible | Boolean | 表示状態（True: 表示, False: 非表示） |
| display_order | Integer | 表示順（昇順で管理） |

### expense_records (経費実績)
| カラム名 | 型 | 説明 |
| :--- | :--- | :--- |
| id | Integer | プライマリキー |
| target_date | Date | 計上日（月次の場合はその月の1日） |
| account_id | Integer | 勘定科目ID (FK) |
| amount | Integer | 金額 |
| unit | String | 計上単位 ('monthly' or 'daily') |
| created_at | DateTime | 登録日時 |
| updated_at | DateTime | 更新日時 |

## 画面一覧
1. **メインメニュー**: 各機能へのナビゲーション。
2. **勘定科目マスタ一覧**: 登録済み科目の一覧表示、表示順変更。
3. **勘定科目登録・編集**: 科目の新規登録および内容更新。
4. **経費一括入力**: 選択した年月/日に基づく一括入力フォーム。
5. **経費データ修正検索**: 修正対象データの検索一覧。
6. **経費データ修正編集**: 個別レコードの修正画面。
7. **経費データ検索・出力**: 月次・日次を分離した実績表示およびCSV出力。

## 運用上の注意
- 経費入力画面には、マスタで「表示」設定かつ設定された「表示順」で科目が表示されます。
- 一度経費データとして使用された勘定科目は削除できません（整合性維持のため）。不要な場合は非表示に設定してください。
