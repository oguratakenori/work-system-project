# 経費管理システム

## 概要
月次および日次の経費実績を管理するシステムです。
勘定科目の管理、経費の一括入力、登録データの修正、および検索・CSV出力機能を提供します。

## 起動方法
以下のコマンドでシステムを起動できます。
ポート番号は `5004` で動作します。

```bash
cd expense-management
python app.py
```

## 主要機能
1. **勘定科目マスタ**: 経費入力に使用する科目の登録・更新・表示順管理。
2. **経費一括入力**: 月次または日次単位で、複数の科目の金額をまとめて登録。
3. **経費データ修正**: 登録済みの経費データの検索および編集。
4. **経費データ検索・出力**: 経費データの条件検索およびCSV出力。月次と日次を分けて表示します。
