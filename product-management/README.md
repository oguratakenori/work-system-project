# 製品管理システム (Product Management System)

## 概要
本システムは、部署ごとの製品マスタ情報を管理するためのWebアプリケーションです。
社員管理システムの部署マスタを参照し、部署単位で独立した製品情報（製品コード、製品名、単価など）を管理します。

## 技術スタック
- **Python / Flask**
- **SQLite / SQLAlchemy** (複数DB連携設定)
- **Bootstrap 5**

## 起動方法
1. `pip install -r requirements.txt`
2. `python app.py`
3. `http://127.0.0.1:5001` にアクセスしてください。
